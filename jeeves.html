Jeeves Command Reference

A comprehensive guide to all available commands and features.

Core Control & Admin Commands

!admin help

Sends a private message with a list of all administrative commands.

!admin reload

Reloads all modules from the disk, applying any code changes.

!admin load <module>

Loads a module into memory by its name (e.g., quest).

!admin unload <module>

Unloads a module from memory.

!admin config reload

Reloads the bot's configuration from its memory (`state.json`).

!admin modules

Lists all modules currently loaded into the bot's memory.

!admin addadmin <nick>

Adds a user to the bot's administrator list at runtime.

!admin deladmin <nick>

Removes a user from the bot's administrator list.

!admin blacklist <module.py>

Adds a module to the runtime blacklist and unloads it.

!admin unblacklist <module.py>

Removes a module from the runtime blacklist.

!admin on|off <module> [#channel]

Enables or disables a loaded module for a specific channel (or the current one).

!admin get <path> [#channel]

Retrieves a configuration value. Use dot notation, e.g., quest.cooldown_seconds.

!admin set <path> <value> [scope]

Sets a config value. Scope can be #channel or global. E.g., !admin set quest.cooldown_seconds 600.

!admin save

Saves the current running configuration to config/config.yaml.saved.

!admin reset

DANGEROUS: Discards all runtime changes and reloads the original config.yaml.

!admin debug on|off

Toggles verbose logging to the debug.log file.

!emergency quit [message]

Forces the bot to disconnect from the server immediately.

User Profile & Courtesy

!gender <identity>

Sets your preferred title (Sir/Madam/Neutral). Accepts terms like "male", "female", "nonbinary", etc.

!pronouns <pronouns>

Sets your preferred pronouns (e.g., he/him, she/her, they/them).

!location <city, country>

Sets your geographical location for modules like Weather and Clock.

!whoami / !profile [user]

Displays the stored preferences for yourself or another user.

!ignore [user]

Makes the bot ignore you. Admins can specify a user to ignore.

!unignore [user]

Makes the bot stop ignoring you. Admins can unignore others.

Fun & Games

!quest [difficulty]

Embark on a solo RPG quest. Difficulty can be easy, normal, or hard.

!quest profile [user]

View your or another player's Quest profile (Level, XP, Class).

!hunt / !hug

Hunt or hug a creature that randomly appears in the channel.

!hunt score [user]

Displays your or another user's hunting and hugging scores.

!adventure

Starts a choose-your-own-adventure where the channel votes on the destination.

!items [user]

Shows the peculiar items you or another user have collected from adventures.

!answer

Answer the service bell when it randomly rings in the channel to score points.

!bell score

Check how many times you've answered the service bell.

!fortune [category]

Receive a fortune. Optional categories: happy, sad, silly, spooky.

!coffee

Serves a beverage appropriate to your local time of day.

!calc <expression>

Performs a mathematical calculation (with occasional whimsical errors).

!pron

Provides a wholesome, helpful scenario with a suggestive trigger.

Utility & Information

!help [command]

Sends a private message with a list of commands, or details for a specific command.

!weather [location/user]

Gets the weather for your set location, or a specified location/user.

!time [location/user]

Gets the local time for your set location, or a specified location/user.

!g <query>

Performs a Google search.

!yt <query>

Searches YouTube for a video.

!ud <term>

Looks up a term on Urban Dictionary.

!wiki <query>

Searches Wikipedia for an article.

!news

Fetches the top news story for your set location.

!tr [lang] <text>

Translates text. If no language code is given, it defaults to English.

!memo <user> <message>

Leaves a message for a user that will be delivered when they next speak.

!shorten <url>

Shortens a long URL.

!gif <query>

Finds a GIF on Giphy that matches the query.

Passive & Ambient Features

Natural Language

Responds to natural questions like "Jeeves, what's the weather?" or "Jeeves, what is 2+2?". Also responds to flirtatious comments and greetings.

URL Title Announcer

Automatically fetches and announces the title of most URLs posted in the chat, including details for YouTube videos.

Sed Corrections

Correct typos in your last message by using sed-style replacement: s/mistake/correction/.

Random Chatter

Periodically posts daily and weekly observations or comments based on channel conversation topics.

Roadtrips

Will occasionally offer to take the channel on a spontaneous roadtrip, a mini-adventure where users can RSVP.
