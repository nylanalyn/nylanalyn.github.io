Jeeves Command Reference

A comprehensive guide to all available commands and features.

Core Control & Admin Commands

!admin <sub-command>

The main command for all administrative functions.

!admin help - Sends a private message with a list of all administrative commands.

!admin reload - Reloads all modules from the disk, applying any code changes.

!admin load <module> - Loads a module into memory by its name (e.g., quest).

!admin unload <module> - Unloads a module from memory.

!admin config reload - Reloads the bot's configuration from its memory (`state.json`).

!admin modules - Lists all modules currently loaded into the bot's memory.

!admin addadmin <nick> - Adds a user to the bot's administrator list at runtime.

!admin deladmin <nick> - Removes a user from the bot's administrator list.

!admin blacklist <module.py> - Adds a module to the runtime blacklist and unloads it.

!admin unblacklist <module.py> - Removes a module from the runtime blacklist.

!admin on|off <module> [#channel] - Enables or disables a loaded module for a specific channel.

!admin get <path> [#channel] - Retrieves a configuration value. Use dot notation, e.g., quest.cooldown_seconds.

!admin set <path> <value> [scope] - Sets a config value. Scope can be #channel or global.

!admin save - Saves the current running configuration to config/config.yaml.saved.

!admin debug on|off - Toggles verbose logging to the debug.log file.

!admin reset - DANGEROUS: Discards all runtime changes and reloads the original config.yaml.

!emergency quit [message] - Forces the bot to disconnect from the server immediately.

User Profile & Courtesy

!gender <identity>

Sets your preferred title (Sir/Madam/Neutral). Accepts terms like "male", "female", "nonbinary", etc.

!pronouns <pronouns>

Sets your preferred pronouns (e.g., he/him, she/her, they/them).

!location <city, country>

Sets your geographical location for modules like Weather and Clock.

!remind <me/user> <timeframe> <message>

Sets a reminder. Timeframe is e.g., in 10 minutes, in 2 hours.

!whoami / !profile

Displays stored preferences.

!whoami - Displays your own stored preferences.

!profile [user] - Displays the preferences for another user.

!ignore / !unignore

Manage the bot's ignore list.

!ignore [user] - Makes the bot ignore you. Admins can specify a user to ignore.

!unignore [user] - Makes the bot stop ignoring you. Admins can unignore others.

Fun & Games

!quest

Embark on a solo RPG quest to fight monsters and level up.

!quest [difficulty] - Difficulty can be easy, normal, or hard.

!quest profile [user] - View a player's Quest profile (Level, XP, Class).

!hunt / !hug

Interact with creatures that randomly appear in the channel.

!hunt or !hug - Captures or befriends the current animal.

!hunt score [user] - Displays hunting and hugging scores.

!release <hunt|hug> - Releases one of your previously captured animals.

!adventure

A choose-your-own-adventure game for the channel.

!adventure - Starts a vote on a destination for the next adventure.

!items [user] - Shows peculiar items collected from adventures.

!bell / !answer

A reaction game to answer the service bell.

!answer - Answer the bell when it randomly rings to score points.

!bell score - Check how many times you've answered the bell.

!fortune [category]

Receive a fortune. Optional categories: happy, sad, silly, spooky.

!coffee

Serves a beverage appropriate to your local time of day.

!calc <expression>

Performs a mathematical calculation (with occasional whimsical errors).

!pron

Provides a wholesome, helpful scenario with a suggestive trigger.

Utility & Information

!help [command]

Sends a private message with a list of commands, or details for a specific command.

!seen <user>

Reports when a user was last seen speaking in the current channel.

!weather [location/user]

Gets the weather for your set location, or a specified location/user.

!time [location/user]

Gets the local time for your set location, or a specified location/user.

!g <query>

Performs a Google search.

!yt <query>

Searches YouTube for a video.

!ud <term>

Looks up a term on Urban Dictionary.

!wiki <query>

Searches Wikipedia for an article.

!news

Fetches the top news story for your set location.

!tr [lang] <text>

Translates text. If no language code is given, it defaults to English.

!memo <user> <message>

Leaves a message for a user that will be delivered when they next speak in that channel.

!shorten <url>

Shortens a long URL.

!gif <query>

Finds a GIF on Giphy that matches the query.

Passive & Ambient Features

Natural Language

Responds to natural questions like "Jeeves, what's the weather?" or "Jeeves, what is 2+2?". Also responds to flirtatious comments and greetings.

URL Title Announcer

Automatically fetches and announces the title of most URLs posted in the chat, including details for YouTube videos.

Sed Corrections

Correct typos in your last message by using sed-style replacement: s/mistake/correction/.

Random Chatter

Periodically posts daily and weekly observations or comments based on channel conversation topics.

Roadtrips

Will occasionally offer to take the channel on a spontaneous roadtrip, a mini-adventure where users can RSVP.
