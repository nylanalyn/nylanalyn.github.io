‚öôÔ∏è Administration

Admin Only
‚ñ∂

Core administrative controls for bot management, module loading, and system configuration

‚ö†Ô∏è All commands in this module require administrative privileges

Quick Commands

!reload
Reload all modules from disk (alias for !admin reload)
!say [#channel] <message>
Make the bot say something in current or specified channel
!emergency quit [message]
Emergency shutdown of the bot (Super Admin only)

Module Management

!admin reload
Reload all modules from disk
!admin load <module>
Load a specific module
!admin unload <module>
Unload a specific module
!admin modules
List all currently loaded modules

Configuration

!admin config reload
Reload config.yaml without restarting the bot

Channel Control

!admin join <#channel>
Make the bot join a channel
!admin part <#channel> [message]
Make the bot leave a channel with optional parting message

System & Debugging

!admin debug <on|off>
Toggle verbose file logging for all modules
!admin debug <module> <on|off>
Toggle debug logging for a specific module

Personal Settings

!gender <identity>

Delete all your profile data from the bot

!unignore [user]

Admin Commands

!setgender <user> <identity>ADMIN
Set another user's gender/title
!setpronouns <user> <pronouns>ADMIN

Natural Language: You can say things like "jeeves, my pronouns are they/them" or "jeeves, I am a woman"

‚öîÔ∏è RPG Quest System

quest.py
‚ñ∂

Comprehensive RPG system with solo quests, group bosses, hardcore mode, prestige, challenge paths, and dungeon runs

Solo Quests

!quest [difficulty]
!q
Go on a solo quest (easy, normal, or hard). Short alias: !qe (easy), !qh (hard)
!quest medic
!qm
Fight monsters for medkit rewards instead of XP
!quest profile [user]
!qp [user]
View your or another player's RPG profile (level, XP, energy, inventory)
!quest class [class_name]
!qc [class_name]
View or set your character class
!quest story
Get a random piece of world lore
!quest top
!quest leaderboard
!qt
View top 10 adventurers by prestige and level

Items & Inventory

!quest search [amount]
!qs [amount]
Search for items (1-20 searches). Find medkits, energy potions, lucky charms, armor shards, and XP scrolls
!quest use <item>
!qu <item>
Use an item from your inventory (medkit, energy_potion, lucky_charm, armor_shard, xp_scroll, dungeon_relic)
!inv
!inventory
!qi
View your items, active effects, and injuries
!medkit [target_user]
Use a medkit to heal injuries. Heal others for 3x XP!

Progression

!quest prestige
Reset to level 1 with permanent bonuses (requires level cap)
!quest prestige challenge
Start a challenge path prestige run with special objectives and rewards
!quest transcend
Transcend beyond max prestige to become a Legend (resets all progress for legendary status)

Hardcore Mode

!quest hardcore [enter|quit|select <item>]
Enter hardcore mode - permadeath challenge! Reach level 50 to earn permanent items. Quit early to abandon run.

Group Content

!mob
Start a group boss fight (opens join window)
!join
Join an active mob encounter
!mob ping <on|off>
Toggle notifications when mob encounters start
!quest boss
Show current boss hunt status and buffs

Abilities

!quest ability [name]
Show or use unlocked abilities from challenge paths (doctor, bloodlust, rally)

Dungeons

!dungeon
Run a ten-room dungeon via private messages (high difficulty, high reward)
!dungeon continue
Continue your dungeon run from a safe haven
!dungeon quit
Abandon your dungeon run and claim partial rewards
!equip
Equip random dungeon gear that can counter threats in dungeon runs

Web Integration

!weblink
Generate a short-lived code to link your IRC account to the quest website for browser play

Admin Commands

!quest reloadADMIN
Reload quest content from JSON files
!quest challenge activate <path>ADMIN
Activate a challenge path
!quest challenge deactivateADMIN
Deactivate the current challenge path
!quest admin boss spawnADMIN
Spawn a new boss for boss hunt
!quest admin boss damage <amount>ADMIN
Deal damage to the current boss
Note: Quest has many short aliases: !q, !qe, !qh, !qp, !qs, !qu, !qm, !qt, !qi for quick access. See the quest website for detailed info on hardcore mode and challenge paths.
