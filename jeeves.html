Jeeves IRC Bot

Command & Module Reference

User & Utility Commands

Courtesy

Manages user profiles, pronouns, and titles.

  • !gender <identity> - Set your preferred title (e.g., sir, madam, neutral).
  • !pronouns <pronouns> - Set your pronouns (e.g., she/her, they/them).
  • !whoami - Display your current profile settings.
  • !profile <user> - Display another user's profile.
  • !forgetme - Delete your courtesy profile.

Convenience

Quick search commands for various services.

  • !g <query> - Perform a Google search.
  • !yt <query> - Search YouTube for a video.
  • !wiki <query> - Search Wikipedia for an article.
  • !ud <term> - Look up a term on Urban Dictionary.
  • !news - Fetch top news headlines for your location.

Location & Time

Provides weather and time based on location.

  • !location <city, country> - Set your default location.
  • !weather - Get weather for your default location.
  • !weather <location/user> - Get weather for a specific place or user.
  • !time - Get the local time for your location.
  • !time <location/user> - Get the time for a specific place or user.

General Bot

Core bot interaction commands.

  • !help - Receive a private message with all commands.
  • !help <command> - Get detailed help for a specific command.
  • !intro - A one-time introduction to the bot's services.
  • !memo <user> <message> - Leave a message for a user.
  • !memos mine - Check messages left for you.

Text Utilities

Tools for text and link manipulation.

  • !shorten <url> - Shorten a long URL.
  • !translate <text> - Translate text to English.
  • !translate <lang_code> <text> - Translate text to a specific language.
  • !translate langs - Get a link to supported language codes.

Fun & Games

Quest

A persistent RPG-style questing game.

  • !quest [difficulty] - Embark on a solo quest (easy, normal, hard).
  • !quest profile [user] - View your or another user's quest profile.
  • !quest story - Get a piece of world lore and your last fight result.
  • !quest class <class_name> - Choose a character class.
  • !mob - Start a group boss fight.
  • !join - Join an active group boss fight.

Hunt

A game of catching animals that randomly appear.

  • !hunt - Hunt the currently active animal.
  • !hug - Befriend the currently active animal.
  • !release <hunt|hug> - Release a previously captured animal.
  • !hunt score [user] - Check your or another user's score.
  • !hunt top - Show the leaderboard.

Adventure

A channel-wide choose-your-own-adventure game.

  • !adventure - Start a new adventure or see the status.
  • 1 or 2 - Vote on the current adventure choice.
  • !adventure last - See the result of the last adventure.
  • !items [user] - See cosmetic items found on adventures.

Bell

A reaction game to answer a service bell.

  • !answer - Answer the bell when it rings to score a point.
  • !bell score - Check your personal score.
  • !bell top - Show the top 5 most attentive users.

Amusements

Miscellaneous fun and flavor commands.

  • !fortune [category] - Get a fortune (happy, sad, silly, spooky).
  • !gif <query> - Find a GIF on Giphy.
  • !coffee - Request a time-appropriate beverage.
  • !pron - Receive a wholesome, helpful scenario.
  • !calc <expression> - Perform a mathematical calculation.

Administrative Commands

Core Control

Essential bot management commands.

  • !admin reload - Reload all modules from disk.
  • !admin config reload - Reload the config from the bot's memory (state).
  • !admin modules - List all currently loaded modules.
  • !admin debug <on|off> - Toggle verbose logging to `debug.log`.
  • !emergency quit [msg] - Immediately shut down the bot.

Configuration Management

Dynamically change bot settings at runtime.

  • !admin on <module> [#channel] - Enable a module for a channel.
  • !admin off <module> [#channel] - Disable a module for a channel.
  • !admin get <module.path> [#channel] - Get a config value.
  • !admin set <module.path> <value> [#channel|global] - Set a config value.
  • !admin save - Writes the current running config to `config.yaml.saved`.
  • !admin reset - DANGEROUS: Reloads config from the original `config.yaml`, discarding all runtime changes.

Channel & Communication

Commands for managing channels and speech.

  • !admin join <#channel> - Make the bot join a new channel.
  • !admin part <#channel> [msg] - Make the bot leave a channel.
  • !say [#channel] <message> - Make the bot say something in the current or a specified channel.
